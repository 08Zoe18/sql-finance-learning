# SQL 财务分析学习

本仓库用于记录 SQL 学习过程，目标：能从数据库取数分析。

## Day 1（2026.03.05）
- 安装 MySQL 8.0
- 配置系统 PATH，验证版本
- 成功登录 MySQL


## Day 2（2026.03.06）
- **练习内容**：
  - 在 `finance_learn` 数据库中练习基础查询：
    - `SELECT` 查看所有数据
    - `WHERE` 筛选金额 > 100 的记录
    - `ORDER BY` 按金额排序
    - 聚合函数 `SUM`、`COUNT` 的使用
- **练习文件**：
  - [`templates/sql_basics_template.sql`](./templates/sql_basics_template.sql) - 包含建表、插入、查询的完整模板
- **收获**：
  - 掌握 SQL 基础查询语法
  - 学会用 Workbench 执行和调试 SQL
  - 整理出可复用的模板文件
